Game server

This HTTP host serves the web client (OTA/assets), privacy policy, and API routes for the game.

Ports (defaults in server.js / admin-server.js):

  • This page — main HTTP, typically port 8082.
  • Database admin — separate process on port 7777 (db-admin.html).
  • WebSocket game — WSS on the port configured for SSL (see server logs).
